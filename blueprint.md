
# Blueprint: 전국 부동산 정보

## 개요

이 프로젝트는 전국 부동산 정보를 한눈에 볼 수 있는 웹 애플리케이션입니다. 초보자도 쉽게 사용할 수 있도록 직관적인 UI와 강력한 검색 및 필터링 기능을 제공합니다.

## 프로젝트 구조

*   `index.html`: 메인 페이지의 HTML 구조
*   `style.css`: 메인 페이지의 스타일
*   `main.js`: 메인 페이지의 기능 및 인터랙션
*   `blueprint.md`: 프로젝트의 설계 및 계획 문서

## 스타일 및 디자인

*   **글꼴**: Noto Sans KR
*   **색상**:
    *   기본 색상: `#2c3e50` (짙은 회색)
    *   강조 색상: `#3498db` (파란색)
    *   배경 색상: `#ecf0f1` (밝은 회색)
*   **레이아웃**:
    *   Flexbox와 CSS Grid를 사용한 반응형 레이아웃
    *   모바일 우선 접근 방식

## 기능

*   **검색**:
    *   지역, 단지명, 지하철역 등 키워드 검색
*   **필터**:
    *   지역 (시/도, 시/군/구)
    *   부동산 유형 (아파트, 오피스텔, 주택 등)
    *   거래 유형 (매매, 전세, 월세)
    *   가격 범위
*   **지도**:
    *   지도에 매물 위치 표시
    *   지도를 통해 매물 검색
*   **매물 목록**:
    *   사진, 가격, 면적 등 주요 정보 표시
    *   무한 스크롤 또는 페이지네이션
*   **상세 정보**:
    *   매물의 상세 정보 페이지

## 현재 계획

### 메인 페이지 UI 구현

*   헤더, 검색창, 필터, 매물 목록, 푸터 등 기본 UI 구성
*   모바일 반응형 디자인 적용
*   `index.html`, `style.css` 파일 수정
